# MVP — Processus · Sistema de Controle Processual

Entrega da Sprint **Gestão Ágil de Projetos e Produtos** (pós-graduação). Planejamento inicial (ideação
e MVP) de um sistema web de **controle processual** para um escritório de advocacia: armazena os dados
dos processos e registra seus **andamentos**, com alertas de prazo e histórico auditável.

> Escopo da atividade: **apenas o planejamento** do produto (não o desenvolvimento).

## 🧩 Produto em uma frase

> Para advogados e gestores que precisam controlar processos e nunca perder prazos, o **Processus** é
> um sistema de controle processual que centraliza processos e andamentos com alertas de prazo —
> diferente de planilhas dispersas, é uma fonte única, auditável e com avisos automáticos.
